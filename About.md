# All In One Transmitter (AIO-TX)



AIO-TX is an open-source, ESP32-C6 based wireless control platform designed for modular robotics, RC systems, and interactive human-interface applications. It supports BLE, Wi-Fi, and ESP-NOW, enabling flexible communication modes ranging from low-latency local control to multi-device wireless networking.



The system features a 0.96” OLED display-based UI, allowing real-time configuration of modes, profiles, and control mappings directly on the device.



Key Features

Multi-protocol wireless communication: BLE, Wi-Fi, ESP-NOW

Built-in ceramic antenna with optional external antenna support for extended range

OLED-based onboard configuration interface (0.96” display)

Modular control profiles for devices such as:

RC vehicles

Robotic systems

Custom ESP-based projects

Wireless Play Mode

Enables local multiplayer interaction between two or more AIO-TX devices using preset game logic for competitive or interactive control experiences.

BLE HID Emulation

Device can function as a wireless keyboard, mouse, or game controller for PCs, TVs, and mobile devices.

Dynamic Input System

12 digital input channels

2 analog input channels

Flexible remapping system allowing digital-to-analog conversion for advanced control behavior

Vision



AIO-TX is designed as a universal wireless control interface, combining embedded systems, human input devices, and multi-protocol communication into a single open hardware platform. It aims to provide a reusable control backbone for future robotics, IoT, and interactive embedded systems.
